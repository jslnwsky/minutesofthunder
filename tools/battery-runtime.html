Use This Correctly

  • Use average current draw, not peak current.
  • Keep reserve in the pack instead of planning to run it flat.
  • Cold weather, gearing, and terrain can reduce runtime fast.
  • LiPo packs should usually stop around 15-20% remaining under practical use.
